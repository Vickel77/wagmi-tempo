# Wagmi Tempo

A web3 application built with Wagmi, React, and Vite for interacting with the Tempo blockchain.

## Setup

### Prerequisites

- Node.js (v18 or higher recommended)
- npm, yarn, or pnpm

### Installation

Install the project dependencies:

```bash
npm install
```

### Running the Development Server

Start the local development server:

```bash
npm run dev
```

The application will be available at `http://localhost:5173` (or another port if 5173 is in use).

Open the URL in your browser to view the application. Make sure you have a wallet extension (like MetaMask) installed to connect.

## Available Scripts

- `npm run dev` - Start the development server
- `npm run build` - Build the project for production
- `npm run preview` - Preview the production build locally
